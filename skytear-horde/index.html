Instructions

Note : As card count in each deck may vary depending on owned content, adjust ALL --card-count variables accordingly.

Print each page on one sheet.
Folding instructions are available there: Youtube - Les ludistes origamistes - [EN] Tutorial #4 : Pocket card holders (2 techniques). Patterns contain folding marks.
Patterns can be customised using variables on top of CSS file:

  • --print-margin (global) : print margin. I recommend a margin of 0 to limit to two cuts and keep two sides straight.
  • --line-width (global) : line thickness of the patterns.
  • --line-color (global) : line color of the patterns.
  • --card-thickness (global) : thickness of a single card (measure thickness of around hundred cards for good accuracy).
  • --card-width (global) : width of the deck.
  • --card-count (deck) : number of cards in the deck. This is used to compute deck thickness.
