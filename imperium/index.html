Bot cards

Files:

  • Antiques
  • Légendes

Instructions

Imprimez les pages en recto-verso.
Une bordure légère délimite les bords des cartes. Toutes les cartes sont de la même taille quelque soit la taille de la table.
Il est possible de personnaliser en modifiant les variables en entête du fichier CSS bot-cards.css :

  • --line-width : épaisseur des contours des cartes
  • --line-color : couleur des contours des cartes

Baggi pockets

Files:

  • Légendes

Instructions

Note : par défaut, le patron imprime l'étiquette sur le recto. Les plis se retrouvent sur le haut et "gêne" lorsque pour insérer les étuis entre eux ; les plis s'insérant les uns dans les autres.

Imprimez chaque page sur une feuille séparée.
Les instructions de pliage sont disponibles ici : Youtube - Les ludistes origamistes - [EN] Tutorial #4 : Pocket card holders (2 techniques). Les patrons contiennent des repères pour le pliage.
Il est possible de personnaliser en modifiant les variables en entête du fichier CSS  :

  • --print-margin (global) : marge d'impression. Je conseille une marge à 0 pour limiter à deux découpes et garder deux côtés bien droits.
  • --line-width (global) : épaisseur des traits des patrons
  • --line-color (global) : couleur des traits des patrons
  • --dim-width (deck) : largeur du deck (également la hauteur de l'étui car le patron est carré)
  • --dim-depth (deck) : épaisseur du deck
  • --title-name (deck) : titre du deck (étiquette)
  • --title-complexity (deck) : complexité du deck (étiquette)
  • --title-name-bg (deck) : couleur de fond de l'étiquette pour le titre (étiquette)
  • --title-name-fg (deck) : couleur du texte de l'étiquette pour le titre (étiquette)
  • --title-complexity-bg (deck) : couleur de fond de l'étiquette pour la complexité (étiquette)
  • --title-complexity-fg (deck) : couleur du texte de l'étiquette pour la complexité (étiquette)
